# Connected Services Spec

## Title: Connected Services

## 1. Description
The "Connected Services" page is the central hub for all external integrations. It allows users to link their Google Account and toggle specific service integrations.

## 2. States

### Not Connected
- Profile photo placeholder.
- "Not Connected" label.
- "Connect Google Account" primary button.
- All integration switches disabled and "Coming Soon".

### Connected
- User's Google Profile photo.
- Display Name and Email visible.
- "Switch Account" and "Sign Out" actions.
- Integration list populated.
