# Archive

Point-in-time handoff notes, AI-to-AI session memos, and status snapshots
from earlier work on NeuroFlow. These reflect the state of the project
*when they were written* — code has moved on since. Kept for history and
for tracing why a decision was made, not as current guidance.

For current, living docs, see the parent `docs/` folder and the root
`README.md` / `CLAUDE.md`.

If you're an AI assistant picking up this project: don't treat anything
in here as the current state of the code. Verify against `lib/` and the
actual commit history first — that's the standard this project holds
itself to (see `RECONCILIATION.md` in this folder for why).
